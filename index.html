Sobre o Projeto


Programa de estágio AWS DevSecOps 2025 - Compass.Uol - Projeto 1

Trata-se de um projeto em que nosso objetivo como estágiarios é prover uma infraestrutura na cloud provider AWS com estrutura de network e uma máquina virtual. Além disso, também expor essa mesma página em um servidor Nginx com um script de monitoramento que notifica ao proprietário da página que a mesma se encontra indisponivel com o http status code, isso através de um webhook da plataforma Discord.

Infraestrutura AWS


VPC

A VPC é rede virtual que se assemelha a uma rede tradicional que você operaria no seu data center, mas com o beneficio da escalabilidade


Subnets

Uma subnet consiste em um intervalo de endereços IP na VPC.


EC2

Uma instância do EC2 é um servidor virtual na Nuvem AWS.


Security Group

Firewall que bloqueia e libera portas sob demanda


CloudWatch + SNS

A Amazon CloudWatch realiza o monitoramento, em tempo real, dos recursos. Analisando performance, integridade e etc. Enquanto o SNS garante a entrega das métricas pro proprietário

Schoolarship


"Em parceria com universidades selecionadas, o Programa de Bolsas oferece oportunidades de aprendizagem para estudantes de tecnologia com desempenho acadêmico excepcional. Oferece bolsas de estudo para faixas de treinamento personalizadas em soluções e estruturas de ponta, com oportunidades potenciais de emprego após a conclusão do programa".
